Set your image as background for header and choose text color.

== Description ==
This plugin will allow you to set up header background image and text color. 

== Installation ==
1. Go to Plugins page
2. Click on Add New button 
3. Search for 'Artistic Header'
4. Click on Install Now button for Artistic Header plugin in search results
5. After plugin is installed click on Activate link for it
6. After plugin activation you will see Header background and Header text color points in menu of Appearance > Customize.
